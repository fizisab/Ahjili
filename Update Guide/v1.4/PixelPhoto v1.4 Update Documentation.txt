1) How To Update

- BACKUP, BACKUP, AND BACKUP everything, files, DATABASE. Please BACKUP!

- Upload "update.php" file located on "Update Guide/v1.4" folder to your server.

- Upload and overwrite ALL files/folders (SELECT ALL) file(s) located inside "Update Guide/v1.4/Script/" folder to your server

- Once the files are uploaded, open and run http://www.YOURSITE.COM/update.php and follow the instructions.

- Note: Don't run "update.php" file until all the files are uploaded.

- Refresh your page, Done !

- For more information, please read our FAQ https://docs.pixelphotoscript.com/
